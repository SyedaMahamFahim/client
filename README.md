# Tech Stack
- React
- Redux
- Axios
- MUI
## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/eCommerce-React-App.git
```

Go to the project directory

```bash
  cd eCommerce-react-app
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
